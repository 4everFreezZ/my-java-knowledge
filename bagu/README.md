## 🚀 如何使用本仓库

1. **制定计划**：首先打开 `TODO List`，了解整个后端面试的知识广度和深度，制定属于你的每日/每周复习计划。
2. **专项突破**：对照 `Java后端开发面试常见八股及对应回答.md` 中的具体知识点，逐个击破。重点理解“为什么”而不是死记硬背。
3. **模拟面试**：可以遮住回答部分，看着问题自己尝试大声回答，锻炼面试时的语言组织与表达能力。
4. **进度追踪**：定期回归 `TODO List` 更新进度，保持学习节奏。
